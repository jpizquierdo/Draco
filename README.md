# Draco
A C++ Arduino library to make a DIY water control system.  
Inspired by a little Canary Islands dragon tree that I brought to Madrid and wanted to monitorize and take care of him properly. Then I thought that I could do something good for a person that would like to build a DIY water control system for vacation or just for have a proper control over some plants.

Materials:  
  Arduino  
  Soil hygrometer like this one. Buy it in aliexpress, ebay, etc.  
    https://www.instructables.com/id/How-to-Use-the-Soil-Hygrometer-Module-Arduino-Tuto/  
  Temperature and relative humidity sensor  
  Real Time Clock (RTC) module (optional, DS1307 for example)  
  Electrovalve
