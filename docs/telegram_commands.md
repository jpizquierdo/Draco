# Telegram Bot Commands

Configure these in @BotFather under "Edit Bot > Edit Commands":

| Command    | Description          |
|------------|----------------------|
| `status`   | System status        |
| `pump`     | Toggle pump          |
| `valve1`   | Toggle valve 1       |
| `valve2`   | Toggle valve 2       |
| `valve3`   | Toggle valve 3       |
| `holidays` | Toggle holidays mode |
